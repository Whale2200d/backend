# 5. 웹프로그래밍을 통한 데이터베이스 처리

## 1. 웹의 필요성

- 비즈니스 업무에서 웹은 커뮤니케이션 채널이 됨.

- 웹이 없는 환경은 더이상 생각할 수 없음
- 웹의 예시
  - 넷플릭스를 통해 영화를 보는 것
  - 인터넷 뱅킹
  - 온라인 수업

- 웹은 모든 비즈니스의 기본이 되고 있음
- 물론, 좀 더 다이나믹하고 온디멘드를 구현하기 위해서는 앱이 있어야 함.

### 1.2. 웹 서버의 필요성

- 서비스를 하기 위해서는 서비스 서버가 필요
- 서버도 특정한 프로그램이 실행되는 컴퓨터를 말함.
- 예를 들어, 데이터베이스 서버는 데이터베이스를 처리하고 관리하는 프로그램임.
- 웹서버는 역시, http 주소를 기반으로 요청한 주소 정보에 대해 이에 대응하는 응답을 줄 수 있는 기능을 수행하는 프로그램임.
- 이러한 프로그램은 컴퓨터에서 실행되는 프로그램임.
- 서버라고 해서 대단한 것이 아니라, 그냥 서비스를 목적으로 만들어진 컴퓨터라고 생각하면 됨.
- 서버라 함은 매일 실행되고 있어야 하며, 죽어서도 안 됨. (가용성)
- 특히, 웹서버의 경우 동시접속에 대해서도 처리가 가능해야 함.
