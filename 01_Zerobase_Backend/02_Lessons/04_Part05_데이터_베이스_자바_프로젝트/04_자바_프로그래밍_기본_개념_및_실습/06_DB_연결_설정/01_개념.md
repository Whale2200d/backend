# 자바 앱을 통한 DB 핸들링

## 6.1. DB 연결 설정

- DB를 연결하기 위해서는 JDBC 드라이버가 필요
- DataGrip or Dbeaver 연결할 때 생각해보면, 5가지 정보를 통해 DB에 접속
- IP, PORT, INSTANCE, USER_ID, PASSWORD

- 예시
  - https://www.naver.com:443/
  - jdbc:DB_VENDER://IP_ADDR:IP_PORT/INSTANCE

## 6.2. Java를 통한 JDBC 프로그램 순서 - (CRUD)

READ

1. JDBC 드라이버 로드: `Class.forName("org.margiadb.jdbc.Driver")`
2. DB Connection 생성: `Connection connection = DriveManager.getConnection(url,dbUser, dbPassword);`
3. SQL을 위한 Statement 객체 생성: `PreparedStatement perparedStatement = connection.prepareStatement(sql);`
4. SQL 문장 실행: `ResultSet resultSet = preparedStatement.executeQuery();`
5. SQL 실행 결과 처리
   ```
   while (rs.next()) {
       System.out.printIn(rs.getString("user_id"));
   }
   ```
6. JDBC 객체들 연결 해제
   ```
   resultSet.close();
   preparedStatement.close();
   connection.close();
   ```

CREATE

1. JDBC 드라이버 로드: `Class.forName("org.margiadb.jdbc.Driver")`
2. DB Connection 생성: `Connection connection = DriveManager.getConnection(url,dbUser, dbPassword);`
3. SQL을 위한 Statement 객체 생성: `PreparedStatement perparedStatement = connection.prepareStatement(sql);`
4. SQL 문장 실행: `int affected = preparedStatement.executeUpdate();`
5. SQL 실행 결과 처리
   ```
   System.out.printf("affected: %d", affected);
   ```
6. JDBC 객체들 연결 해제
   ```
   preparedStatement.close();
   connection.close();
   ```
