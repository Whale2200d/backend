## 2.1. 상수 변수 데이터형

- 상수
  - 변하지 않거나 일정한 값을 가진 “수 or 문자 or 문자열 or 데이터”
  - 선언과 동시에 초기화 or 선언 후 다음에 초기화 (1번만 가능)
  - 한번 초기화되면, 더이상 변경 이슈가 없는 값들을 보통 상수로 표현함.
  - 예시 (Sample01.java)

    ```sql
    // 선언과 동시에 초기화(할당)
    final int MONTH = 12;

    // 선언하고 초기화(할당)를 따로하는 방법
    final int DAY;

    public Sample(int day) {
    	this.DAY = day;
    }

    // 문자열
    final String ERROR_MESSAGE = "에러가 발생했습니다.";
    ```

- 변수
  - 데이터 처리 혹은 프로그래밍을 하기 위해서는 값을 변경하거나 저장할 수 있는 공간이 필요
  - 값을 저장할 수 있는 메모리 공간
  - 이러한 공간은 계속해서 변경될 수 있지만, 최종적으로 사용한 값만 가지고 있음.
  - 변수는 데이터 형태가 필요함(숫자를 저장하기 위한 값인지 문자를 저장하기 위한 값인지)
  - 예시 (Sample02.java)

    ```sql
    // 변수타입(형태) 변수이름(의미있는_이름);
    String myName;
    int 내_나이값;
    int 내1학기점수;
    int int_my_age;

    int myAge;
    int MyAge;

    int 나이;
    double 키;
    double Weight;
    boolean 사용여부;
    char 코드 = 'A';
    ```

    - 기본타입(프리미티비 타입)
      - 숫자
        - 정수형: byte, short, int, long
        - 실수형: float, double
      - 불린형: boolean
      - 문자형: char
    - 참조형(객체형)
      - 선언가능한 모든 클래스 형태 (String, Date, LocalDateTime, Scanner 등)
    - 변수명을 짓는 규칙 존재
      - 영문자 대소문자가 가능하면 길이제한 없음
      - 예약어로 지정된 단어 사용 불가(int, for, if 등)
      - 숫자로 시작할 수 없음
      - 특수문자는 ‘\_’와 ‘$’만 가능
      - 필수는 아니지만 명명규칙 존재(camelCase, PascalCase)
