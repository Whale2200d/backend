# Test 2

## 문제 설명

두미는 영단어를 열심히 공부하고 있다. 그런데 두미는 단어의 시작과 끝은 잘 기억이 나지 않아서 곤란해 하고 있다.

두미의 절친인 당신은 두미를 위한 프로그램을 만들어 주려고 한다. 만들어주려 하는 단어 검색 프로그램은 아래와 같다.

- 단어의 검색은 words 배열 내에서 수행한다.
- 검색하고자 하는 단어들은 queries 배열로 주어진다.
  - 각 검색 단어는 단어의 앞부분 또는 뒷부분 일부만으로 구성되어 있다.
  - 앞부분 일부만으로 구성된 경우 단어 마지막에 `*`을 붙인다. (ex. `"hel*"`, `"goo*"`)
  - 뒷부분 일부만으로 구성된 경우 단어 시작에 `*`을 붙인다. (ex. `"*able"`, `"*shed"`)
  - `*`의 위치에는 길이와 내용이 상관없이 어떤 문자열이 와도 매칭 대상이 된다.

프로그램의 출력은 각 검색 결과에 해당하는 단어의 개수를 배열로 반환한다.

상술한 프로그램을 구현하라.

## 입력 설명

- `0 < words.length <= 10000`
- `0 < words[i].length <= 1000`
- `0 < queries.length <= 10000`
- `0 < queries[i].length <= 1000`

## 출력 설명

각 매칭 개수를 나열한 정수의 배열

## 입출력 예시

- 입력
  - `words = {"hello", "hell", "good", "goose", "children", "card", "teachable"}`
  - `queries = {"hell*", "goo*", "*able", "qua*"}`
- 출력 : `{2, 2, 1, 0}`
