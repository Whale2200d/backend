# Test 3

## 문제 설명

윤식이는 검색 엔진 개발자로, 단어를 검색하는 프로그램을 만들어 달라는 의뢰를 받았다.

윤식이의 클라이언트는 아래와 같은 방식으로 단어를 검색하고 싶어 한다.

- `words`에 검색 대상이 되는 단어 풀이 배열로 주어진다.
- `queries`에 검색하고자 하는 검색어들의 배열로 주어진다.
  - 검색하고자 하는 입력은 `"[알파벳][? 여러개]"`로 주어진다.
  - 예를 들어, `"hel??"`로 검색할 경우, `"hello"`는 검색 대상이 되지만, `"hell"`은 검색 대상이 되지 않는다.
  - 검색어의 검색 결과는 검색 대상이 되는 모든 단어를 `words`에 나타난 순서대로 하는 배열이다.

프로그램의 출력은 각 검색어에 대한 결과(문자열 배열)을 모아 2차원 배열로 반환한다.
윤식이가 의뢰받은 프로그램을 구현하라.

## 입력 설명

- `0 < words.length <= 10000`
- `0 < words[i].length <= 1000`
- `0 < queries.length <= 10000`
- `0 < queries[i].length <= 1000`
- 검색어에서 `?`는 반드시 한 개 이상 존재한다.

## 출력 설명

검색 결과를 2차원 배열로 출력

## 입출력 예시

- 입력
  - `words = {"hello", "hear", "hell", "good", "goose", "children", "card", "teachable"}`
  - `queries = {"he??", "g???", "childre?", "goo????", "?"}`
- 출력
  - `{{"hear", "hell"}, {"good"}, {"children"}, {}, {}}`
