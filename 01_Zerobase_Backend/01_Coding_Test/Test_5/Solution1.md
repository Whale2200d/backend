# Test 1

## 문제 설명

문자열 s가 있을 때, 이 문자열을 재배치하여 만든 문자열을 '애너그램'이라고 한다.
예를 들어, `"fine"`은 `"infe"`의 애너그램이라고 할 수 있다.
s가 영문 소문자로만 이루어져 있다고 할 때,문자열 t가 문자열. s의 애너그램인지 판단하는 프로그램을 작성하라.

## 입력 설명

- 0 <= s.length <= 10000
- 0 <= t.length <= 10000

## 출력 설명

- 애너그램 여부를 논리값으로 반환

## 입출력 예시

- 입력 : s = "imfinethankyou", t = "atfinemnoyuki"
- 출력 : true
