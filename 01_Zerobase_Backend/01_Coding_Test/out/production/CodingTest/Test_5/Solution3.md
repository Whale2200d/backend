# Test 3

## 문제 설명

정수로 이루어진 리스트 arr에서 피크의 인덱스 peak를 다음과 같이 정의하자.

- 0 < peak < arr.length - 1
- x[peak - 1] < x[peak]
- x[peak + 1] < x[peak]

주어진 리스트 arr에서 피크가 최대 한 개 존재할 때, 피크의 인덱스 peak를 출력하라.

단, 배열에 피크가 존재하지 않을 경우 -1을 출력하라.

## 입력 설명

- 3 < arr.length < 1000000
- i <= peak 일 경우, arr[i - 1] < arr[i]
- i >= peak 일 경우, arr[i + 1] < arr[i]
- 피크가 존재하지 않을 경우, arr는 단조증가 또는 단조감소. 즉, 모든 i에 대해 arr[i - 1] < arr[i] 또는 모든 i에 대해 arr[i + 1] < arr[i]

## 출력 설명

배열의 유일한 피크값의 인덱스를 정수로 반환

## 입출력 예시

- 입력 : arr = [-3, 0, 3, 4, 5, 12, 15, 14, 12, 11]
- 출력 : 6
