# 배당금 프로젝트

## 1. 주제 설명

### 주식

- 회사의 자본을 구성하는 단위
- 기업의 자본금 = 1주당 액면가 \* 발행주 수

### 배당금

- 회사의 이익을 주주들에게 배분하는 금액
- 배당수익률: N/A (배당금을 지급하지 않는 주식)
- 배당금은 회사마다 주당비율, 지급날짜, 지급기간 모두 다름.

### 주식 배당금 API

- 배당금을 배분하고 있는 주식의 정보를 제공하는 API 서비스를 구현한다.
  - 코카콜라는 얼마의 배당금을 줬을까
  - 나이키의 배당금 추이는 어떻게 될까
  - 애플은 언제 배당금을 줬을까

## 2. 요구사항 정리

- 한 줄 정리: 내가 원하는 회사의 배당금 정보를 보고 싶다.

1. [회사]를 저장/삭제 관리할 수 있는 기능
   - 회사 정보를 저장하는 API
   - 저장된 회사 정보를 삭제하는 API
   - 회사 정보를 관리하기 위한 DB 테이블 설계

2. 회사에 해당하는 [배당금] 정보를 제공하는 기능
   - 회사의 배당금 정보를 조회하는 API
   - 회사의 배당금 정보를 스크래핑
   - 회사 배당금 정보를 관리하기 위한 DB 테이블 설계

3. [내]가 원하는
   - 회원가입 및 로그인/로그아웃 API
   - 사용자 인증

### 스크래핑

- 왜 스크래핑을 하는가?
  - 활용 분야가 다양함.
  - 쇼핑몰, 금융, 업무 자동화 등등

- 서비스 부하를 줄이기 위한 캐시 서버 구성하기
